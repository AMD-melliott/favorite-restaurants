# Favorite Restaurants

Welcome to our collaborative restaurant recommendation list!

## Workshop Goal
Practice the Git/GitHub workflow by contributing your favorite restaurant to our shared list.

## Getting Started
1. Fork this repository
2. Clone it to your computer
3. Create a new branch
4. Add your restaurant recommendation
5. Push your changes
6. Create a Pull Request

See the workshop materials for detailed step-by-step instructions.
